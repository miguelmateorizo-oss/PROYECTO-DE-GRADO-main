  Figura 5. Interfaz “Mi Bandeja Personal”.
Nota. Elaboración propia.
La imagen muestra la interfaz de "Mi Bandeja Personal" del prototipo SIGECOR, donde se visualizan los documentos que han sido asignados específicamente a un dinamizador.
La interfaz permite visualizar de manera agrupada los documentos según su estado de gestión (Por Atender, En Proceso, Finalizadas) y acceder a cada uno para dar respuesta o realizar el seguimiento correspondiente. Esta interfaz da cumplimiento directo a la Historia de Usuario HU-005.
